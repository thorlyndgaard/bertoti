# Meu Projeto

Este projeto é uma aplicação full-stack utilizando **React** no front-end e **Spring Boot** no back-end, com **Maven** como gerenciador de dependências.

## Tecnologias Utilizadas

- **Front-end:** React
- **Back-end:** Spring Boot
- **Gerenciador de dependências:** Maven
- **Linguagem:** Java (back-end), JavaScript/TypeScript (front-end)

## Pré-requisitos

Certifique-se de ter instalado em sua máquina:

- Java 17 (ou superior)
- Maven
- Node.js e npm

## Como Rodar o Projeto

### 1. Rodando o Back-end (Spring Boot)

No terminal, dentro da pasta do back-end, execute:

```bash
./mvnw clean spring-boot:run
